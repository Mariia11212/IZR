Проєкт складається з двох основних логічних частин:
1.  **Власні структури даних:** Реалізація базових колекцій, таких як динамічний масив, зв'язані списки тощо.
2.  **Геометрична бібліотека:** Набір класів для представлення 2D-фігур (точка, вектор, прямокутник, коло) та виконання над ними афінних перетворень (переміщення, поворот, масштабування).

## Юніт-тестування

Основна мета цієї лабораторної роботи — створення комплексного набору юніт-тестів для перевірки коректності реалізованих модулів.

### Фреймворк
Тести розроблено з використанням фреймворку **NUnit**, який є одним зі стандартів для тестування у .NET-екосистемі.

### Покриті модулі
На даний момент юніт-тестами покрито наступні ключові модулі проєкту:

*   **`MyDynamicArrayBasedList<T>`**:
    *   Створення та ініціалізація списку.
    *   Додавання, вставка та видалення елементів (`Add`, `InsertAt`, `RemoveAt`).
    *   Доступ до елементів та їх пошук (`GetAt`, `IndexOf`, `Contains`).
    *   Автоматичне розширення ємності масиву.
    *   Очищення списку (`Clear`) та робота ітератора.
    *   Перевірка граничних випадків (невалідні індекси, робота з порожнім списком).

*   **`Vector`**:
    *   Коректність створення та ініціалізації.
    *   Обчислення довжини (магнитуди) вектора.
    *   Операція масштабування (`Scale`).

*   **`Point`**:
    *   Коректність створення та ініціалізації.
    *   Обчислення відстані до іншої точки (`GetDistanceTo`).
    *   Операція переміщення за допомогою вектора (`Translate`).

*   **`Translation`**:
    *   Застосування трансформації переміщення до точки.
    *   Перевірка роботи з нульовим вектором.

*   **`Rectangle`**:
    *   Валідація вхідних даних у конструкторі (ширина та висота мають бути додатними).
    *   Обчислення площі (`GetArea`) та периметра (`GetPerimeter`).
    *   Застосування трансформацій.

*   **`Circle`**:
    *   Валідація вхідних даних (радіус має бути додатним).
    *   Обчислення площі та периметра.
    *   Перевірка належності точки колу (`ContainsPoint`).
    *   Застосування трансформацій (переміщення та масштабування).
